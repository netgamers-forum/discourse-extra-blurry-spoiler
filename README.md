# Extra Blurry Spoiler

A Discourse theme component that strengthens the blur effect of the [Spoiler Alert](https://meta.discourse.org/t/discourse-spoiler-alert/12650) plugin.

## What it does

- Increases spoiler blur to `48px` (default is `~16px`)
- Locks the blur on hover/focus to `32px` so accidental reveals are avoided on desktop

## Installation

1. Push this repo to GitHub
2. In Discourse admin: **Customize → Themes → Install → From a git repository**
3. Add it as a component to your active theme
